image_converter_1_0 - program for converting image files

This python program uses a Tkinter GUI to display a menu to select the input and output folders, and the output image format. Once the folders and format are selected, the program loops over all files in the input folder and checks if they have any of the listed image extensions. If they do, the program opens the image using Image.open() and saves it with the new format using the .save() method.
